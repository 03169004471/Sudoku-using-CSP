# Sudoku Game using Constraint Satisfaction Problem (CSP)

## Overview

This GitHub project presents an interactive Sudoku game, implemented using Python's Tkinter library and the Constraint Satisfaction Problem (CSP) approach. It offers a visually appealing and user-friendly interface for playing Sudoku, a popular puzzle game. The project is titled "Sudoku 9x9" and features a black-themed GUI with colorful accents, contributing to an engaging user experience.

## Features

1. **Graphical User Interface**: Built with Tkinter, the interface includes a 9x9 Sudoku grid, buttons for various actions, and a progress bar, all styled with a black background and white text for a modern look.

2. **Sudoku Grid**: The core of the game, this grid allows users to input numbers and attempt to solve the Sudoku puzzle. The cells have a pleasant blue background with black font, and thicker borders are used to demarcate the 3x3 subgrids.

3. **Interactive Buttons**:
   - **Solve using AI**: Solves the current Sudoku puzzle using CSP.
   - **Check**: Validates the current solution.
   - **New Game**: Resets the board for a new game.

4. **CSP Implementation**: The game logic utilizes the CSP approach to solve Sudoku puzzles. It enforces all standard Sudoku rules, including unique numbers in each row, column, and 3x3 subgrid.

5. **Customizable Puzzle Generation**: The initial board is pre-filled with some numbers, and additional puzzles can be generated by randomly removing values.

## Technical Details

- **Language and Libraries**: Python, Tkinter for GUI, and the `constraint` library for implementing CSP.
- **Modular Design**: The code is structured as a `SudokuGame` class, encapsulating all functionalities and making it easy to understand and modify.
- **User Input Handling**: Users can input numbers directly into the grid, and the game validates these inputs against the Sudoku rules.
- **Constraint Handling**: The CSP solver checks for all Sudoku constraints, including row, column, and subgrid uniqueness.

## Usage

To run the game, simply clone the repository and execute the main Python script. Ensure that Python and the required libraries (`tkinter` and `constraint`) are installed.

This project offers an excellent example of combining GUI programming with algorithmic problem-solving. It's ideal for anyone interested in graphical applications, game development, or exploring CSP algorithms. Whether you're a Sudoku enthusiast or a Python developer, this project provides a robust and enjoyable platform to play and learn.
